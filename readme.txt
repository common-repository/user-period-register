== Description ==

= Register users with time limit =
* Register users with time limit.
* Can temporarily register users.

= Cooperation with other plugins =
* Work with [User Mail Only Register](https://wordpress.org/plugins/user-mail-only-register/).
* Work with [Software License Manager Client](https://wordpress.org/plugins/software-license-manager-client/).
* The role can be changed from the user by charging.

== Installation ==

1. Upload `user-period-register` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
